# Security Policy

## Supported Versions

`conflictcast` is currently supported on the latest `0.1.x` release line.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Instead:

1. Email the maintainers with a clear reproduction and impact summary.
2. Include affected versions, configuration details, and any suggested mitigations.
3. Allow time for triage and coordinated disclosure before publishing details.

We aim to acknowledge reports within 3 business days and will share remediation
status as the investigation progresses.
